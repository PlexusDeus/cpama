Write a program that declares several `int` and `float` variables—without initializing them—and then prints their values. Is there any pattern to the values? (Usually there isn’t.)

---
Program:
```c
#include <stdio.h>

int main(void) {
  int a, b, c;
  float x, y, z;

  printf("Value of a: %d\n", a);
  printf("Value of b: %d\n", b);
  printf("Value of c: %d\n", c);
  printf("Value of x: %g\n", x);
  printf("Value of y: %g\n", y);
  printf("Value of z: %g\n", z);
}
```

Output:
```
Value of a: -542823696
Value of b: 29536
Value of c: 1827832528
Value of x: 4.59121e-41
Value of y: 2.34543e+27
Value of z: 4.59121e-41
```
