Show the output produced by each of the following program fragments. Assume that `i` and `j` are `int` variables.

1. `i = 5;
j = ++i * 3 - 2;
printf("%d %d", i, j);`
2. `i = 5;
j = 3 - 2 * i++;
printf("%d %d", i, j);`
3. `i = 7;
j = 3 * i-- + 2;
printf("%d %d", i, j);`
4. `i = 7;
j = 3 + --i * 2;
printf("%d %d", i, j);`

---

1. `6 16`
2. `6 -7`
3. `6 23`
4. `6 15`
