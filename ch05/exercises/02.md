The following program fragments illustrate the logical operators. Show the output produced by each, assuming that `i`, `j`, and `k` are `int` variables.

1. `i = 10; j = 5;
printf("%d", !i < j);`
2. `i = 2; j = 1;
printf("%d", !!i + !j);`
3. `i = 5; j = 0; k = -5;
printf("%d", i && j || k);`
4. `i = 1; j = 2; k = 3;
printf("%d", i < j || k);`

---

1. `1`
2. `1`
3. `1`
4. `1`
