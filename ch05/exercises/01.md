The following program fragments illustrate the relational and equality operators. Show the output produced by each, assuming that `i`, `j`, and `k` are `int` variables.

1. `i = 2; j = 3;
k = i * j == 6;
printf("%d", k);`
2. `i = 5; j = 10; k = 1;
printf("%d", k > i < j);`
3. `i = 3; j = 2; k = 1;
printf("%d", i < j == j < k);`
4. `i = 3; j = 4; k = 5;
printf("%d", i % j + i < k);`

---

1. `1`
2. `1`
3. `1`
4. `0`
